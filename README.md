# Metronic Template

## Installation

1. clone metronic-template here git@github.com:WebAppDevTech/metronic-template.git

*include submodules
-CMS
-UCS
-MPIS
-MPF
-ORDDERPUB
-PAIS-TEMPLATE
2. npm install
3. makesure all repos are checkout as main
4. npm run serve
5. eat your favorite junkfood and wait while building
